# Guía: Crear Collections en Shopify Admin

## Paso 0: Crear la Collection automática

1. Ve a **Products > Collections > Create collection**
2. **Title**: `Collections`
3. **Collection type**: Automated
4. **Condition**: Product tag **is equal to** `collection-set`
5. Guarda
